# Raya Ayurveda Color Palette

## New Brand Colors

| Color Name | Hex Code | Usage |
|-----------|----------|-------|
| **Primary Purple** | `#5E17EB` | Main brand color, buttons, links, accents |
| **Light Gray** | `#F4F4F4` | Light backgrounds, subtle sections |
| **Cream** | `#FFFBF7` | Primary backgrounds, cards |
| **Beige** | `#EAE9E3` | Testimonials, secondary backgrounds |
| **Dark** | `#181818` | Text, navigation, footer |

## Color Applications

### Navigation
- **Background**: Dark (#181818) with gradient and transparency
- **Links**: White with purple hover (#5E17EB)

### Footer
- **Background**: Dark (#181818)
- **Accent headings**: Primary Purple (#5E17EB)

### Home Page
- **Conditions Section**: Primary Purple background (#5E17EB)
- **Testimonials**: Beige background (#EAE9E3)
- **Instagram Section**: Purple gradient (from #5E17EB to #9D6BF7)

### Buttons
- **Primary Button**: Purple (#5E17EB) with darker hover (#4B12BD)
- **Secondary Button**: Outlined with purple border

### Status & States
- **Default**: Primary Purple (#5E17EB)
- **Hover**: Darker Purple (#4B12BD)
- **Focus**: Purple ring (#5E17EB)

## Notes

- Legacy green colors have been mapped to purple for backward compatibility
- All `green-700`, `green-800` → `#5E17EB` or `#4B12BD`
- All `green-50`, `green-100` → `#EAE9E3` or `#F4F4F4`
- Testimonial cards use beige background (#EAE9E3) for subtle contrast
